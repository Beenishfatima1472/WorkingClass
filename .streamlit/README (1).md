# models/

This folder holds the trained Fake-Fatwa Shield model weights.

**These files are NOT committed to GitHub** (see `.gitignore`).

They are generated by running `save_model.py` against the private training notebook,
which is withheld pending journal publication.

## Required files (generated locally)

| File | Description |
|---|---|
| `isolation_forest.pkl` | Trained IsolationForest anomaly detector |
| `tfidf_vectorizer.pkl` | Fitted TF-IDF vectorizer |
| `authentic_texts.json` | Authenticated corpus used for cosine similarity |
